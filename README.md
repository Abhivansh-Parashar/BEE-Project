# Node + React Project

This is a full-stack project utilizing a React frontend and a Node.js backend.

## Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your system.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Abhivansh-Parashar/BEE-Project.git
   cd BEE-Project
   ```

2. **Set up the Backend:**
   Open a terminal and navigate to the backend directory to install its dependencies and start the server.
   ```bash
   cd backend
   npm install
   npm start # or node server.js
   ```

3. **Set up the Frontend:**
   Open a *new* terminal window and navigate to the frontend directory to install its dependencies and start the React application.
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
